* ARM SMMU V3 PMU Bindings

This is ARM SMMUv3 PMU (Performance Monitoring Unit) module.

Required properties:
- compatible		: Shall be "arm,smmu-v3-pmcg"
- SMMU_ADDR:		: reference to smmuv3 device node
- interrupt-parent	: reference to interrupt parent device node
- interrupts		: interrupt number generated by the control block
- interrupt-names	: Shall be "combined"

Optional properties:
- override_cfg		: override parameter for SMMUV3 PMCG CFGR register
- override_ceid		: override parameters for SMMUV3 PMCG CEID registers

Example:
	smmu_v3_pmcg {
		compatible = "arm,smmu-v3-pmcg";
		SMMU_ADDR = <&smmu>;
		override_cfg = <0x00c01f03>;
		override_ceid = <0x0 0x7f 0x0 0x0>;
		interrupt-parent = <&gic>;
		interrupts = <GIC_SPI 150 IRQ_TYPE_EDGE_RISING>;
		interrupt-names = "combined";
	};
